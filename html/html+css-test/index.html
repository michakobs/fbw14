Tasks

  1. Make and link a CSS stylesheet and work from there.
  2. Align the text in the header to the center of the page.
  3. Change the color of THIS list item to blue.
  4. Images Task
    • Find the images section in the HTML file.
    • Find the div with the class of "background" and select it by its class name in CSS.
    • Give the div with the class of "background" a width and height.
    • Give the div with the class of "background" a background image using the image "tree.jpg". The image can be found in the "images" folder
    • Set the background image to cover the div.
    • Make sure the background image does not repeat itself
    • Position the background image at the center of the div.
  5. Make all of the images in the "images section" appear side-by-side.
  6. Select the second image (mountain image) by its alt attribute value. Give the image padding and a solid blue border.
  7. Find the div with 10 span elements nested inside. Change the background color of every even span element using nth-child().
  8. Media Query Task
    • Go to the section titled "Media Queries". Change the background color and size of the the box with the class of "mediaquery". It is in the section titled "media query".
    • Mobile background color: blue, size: 5rem x 5rem
    • Tablet background color: yellow, size: 7rem x 7rem - BREAKPOINT: 768px.
    • Desktop background color: red, size: 10rem x 10rem - BREAKPOINT: 1024px.
    • Large Desktop background color: purple, size: 12rem x 12rem BREAKPOINT: 1440px
  9. Positioning Task
    • Go to the section titled "Positioning"
    • Give the div with the class "relative" a relative position. Position this div on the top-right hand side of its parent element (the box)
    • Give the div with class "absolute" an absolute position. Position this div on the bottom-left hand side of its parent element (the relative container).
    • Make the header element fixed to the top of the page.
  10. If the header element hides part of the task section, add margins where needed.
  11. Flexbox Task
    • Go to the section titled "Flexbox"
    • Align the "items" in the "container" in a row.
    • Make the items take up multiple rows.
    • Center the rows in the container.
    • Make the items take up all available space on the last row.
  12. Grid Layout Task
    • Go to the section titled "Grid Layout"
    • Make the div with the class "wrapper" a grid
    • Give it three columns and four rows of equal size.
    • Give the wrapper a border
    • Give the grid a grid gap of 5px
    • Give all cells a border.
    • Make the second cell twice as high as the rest of the cells spanning row lines 2 - 4.
    • Make the third cell twice as wide and twice as high as the the rest of the cells spanning column lines 2 - 4 and rows lines 1 - 3
    • Make the seventh cell twice as wide as the rest of the cells spanning column lines 2 - 4
  13. Give three h3 elements a class, change the font color, font size and font family of the class.
  14. Select all the anchor links in the footer. Select them as DESCENDANTS of the footer. Change the color of the footer links on :visited and :hover.
  15. When you have finished all the tasks, strike out ALL the TASK list items.
